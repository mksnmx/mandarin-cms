1С-Битрикс Малый Бизнес: Управление сайтом 17.5.12

1. На сервере в каталоге /bitrix/modules/ создать каталог mandarinbank.mandarinbank

2. Распаковать архив, из каталога .last_version скопировать две папки (install и lang) на сервер в каталог /bitrix/modules/mandarinbank.mandarinbank - в этом каталоге должны появиться две папки (install и lang)

3. В администрирование сайта, раздел Marketplace/Установленные решения (bitrix/admin/partner_modules.php) - найти строку mandarinbank.mandarinbank, действие - Установить. 
4.Модуль установлен.



Настройка модуля:


1. В администрирование сайта, раздел Магазин/Настройки/Платежные системы (bitrix/admin/sale_pay_system.php), нажать "Добавить платежную систему"

2. В поле обработчик выбрать: Раздел пользовательские, платёжная система mandarinbank

3. Вписать заголовок, описание и логотип по желанию, есть в папке с модулем, тип оплаты Безналичный

4. Ниже на вкладке "По умолчанию", заполнить поля:


Сумма к оплате: Заказ : Стоимость заказа
Номер заказа: Заказ : Код заказа(ID)

Email покупателя: Пользователь : Электронный адрес

Секретный ключ : Значение : Ваш секретный ключ

ID кошелька : Значение : Ваш merchant ID


Сохранить.
