== Description ==

Плагин Mandarin Payment Gateway обеспечивает интеграцию вашего интернет-магазина с платежной системой Mandarin. Версия 2.0 полностью переработана для совместимости с современными версиями WordPress и WooCommerce, включая полную поддержку нового блочного интерфейса checkout.

**Ключевые особенности:**

Плагин поддерживает все современные способы оплаты через платформу Mandarin, включая банковские карты Visa и MasterCard, электронные кошельки Яндекс.Деньги и WebMoney, а также терминалы QIWI. Интеграция происходит через безопасное перенаправление на сертифицированные серверы Mandarin, что обеспечивает максимальную защиту данных ваших клиентов.

Современный пользовательский интерфейс включает элегантный индикатор загрузки с автоматическим перенаправлением на платежную страницу. Это создает плавный и профессиональный опыт оплаты, который внушает доверие покупателям и снижает количество брошенных корзин.

Плагин полностью совместим как с классическим checkout через шорткоды, так и с новым блочным интерфейсом WooCommerce Blocks. Это означает, что он будет работать независимо от того, какую версию интерфейса checkout использует ваш магазин.

**Безопасность и надежность:**

Все транзакции обрабатываются с использованием криптографических подписей SHA-256, что исключает возможность подделки платежных данных. Система автоматической верификации callback-уведомлений обеспечивает мгновенное обновление статусов заказов при успешной оплате.

Плагин включает расширенную систему логирования для диагностики проблем, что значительно упрощает техническую поддержку и интеграцию в сложных конфигурациях хостинга.

**Техническая поддержка:**

Данная версия плагина полностью протестирована с современными версиями PHP, WordPress и WooCommerce. Код соответствует актуальным стандартам разработки и использует современные API WordPress и WooCommerce.

== Installation ==

**Системные требования:**
- WordPress 6.0 или выше
- WooCommerce 8.0 или выше
- PHP 7.4 или выше
- Активированный SSL-сертификат (рекомендуется для безопасности)

**Пошаговая установка:**

Процесс установки занимает всего несколько минут и не требует специальных технических знаний. Начните с получения учетных данных в личном кабинете Mandarin, затем следуйте инструкциям ниже.

**Шаг 1: Получение учетных данных**
Зарегистрируйтесь в системе Mandarin и получите ваши уникальные Merchant ID и Merchant Secret в личном кабинете. Эти данные необходимы для установления безопасного соединения между вашим магазином и платежной системой.

**Шаг 2: Установка плагина**
Загрузите архив плагина через стандартный интерфейс WordPress: Плагины → Добавить новый → Загрузить плагин. Альтернативно, можете распаковать архив в папку /wp-content/plugins/ на вашем сервере.

**Шаг 3: Активация**
Перейдите в раздел "Плагины" панели администратора WordPress и активируйте плагин "Mandarin Payment Gateway". После активации плагин автоматически зарегистрируется как доступный способ оплаты в WooCommerce.

**Шаг 4: Настройка**
Откройте WooCommerce → Настройки → Платежи и найдите раздел "Mandarin". Нажмите "Настроить" и заполните обязательные поля:
- Merchant ID: ваш уникальный идентификатор магазина
- Merchant Secret: секретный ключ для генерации подписей

Убедитесь, что опция "Включить данную систему" отмечена галочкой.

**Шаг 5: Тестирование**
Создайте тестовый заказ в вашем магазине, выберите способ оплаты "Mandarin" и убедитесь, что происходит корректное перенаправление на платежную страницу. При успешном тестировании система готова к приему реальных платежей.

== Frequently Asked Questions ==

= Работает ли плагин с новым блочным checkout в WooCommerce? =

Да, версия 2.0 полностью поддерживает как классический checkout, так и новый блочный интерфейс WooCommerce Blocks. Плагин автоматически определяет тип используемого checkout и адаптируется соответствующим образом.

= Безопасно ли использовать данный плагин? =

Абсолютно безопасно. Все конфиденциальные данные покупателей обрабатываются исключительно на сертифицированных серверах Mandarin. Ваш сайт лишь инициирует безопасное перенаправление и получает зашифрованные уведомления о статусе платежа.

= Что делать, если плагин не отображается в списке способов оплаты? =

Убедитесь, что вы используете WordPress 6.0+ и WooCommerce 8.0+. Проверьте, что плагин активирован и в настройках указаны корректные Merchant ID и Secret. При сохраняющихся проблемах включите режим отладки WordPress для анализа логов.

= Поддерживаются ли возвраты через плагин? =

В текущей версии возвраты обрабатываются через личный кабинет Mandarin. Функциональность автоматических возвратов планируется к добавлению в будущих версиях плагина.

= Можно ли настроить внешний вид страницы оплаты? =

Страница предварительной загрузки использует адаптивный дизайн, который автоматически подстраивается под стиль вашего сайта. Дополнительная кастомизация возможна через CSS или обращение к разработчикам плагина.
